## Task 2: Implement exponential backoff in JobQueue.fail()

**Files:** `src/worker/queue.py`

### Context

Currently `fail()` resets job to `pending` with no delay. Need to compute `next_retry_at` based on attempt count.

### Action

Modify the retry branch in `fail()`:

```python
# Exponential backoff: base=30s, multiplier=2, cap=3600s
base_delay = 30  # seconds
max_delay = 3600  # seconds (1 hour)
attempt = job.attempt_count  # already incremented
delay = min(base_delay * (2 ** (attempt - 1)), max_delay)
next_retry_at = datetime.utcnow() + timedelta(seconds=delay)

job.next_retry_at = next_retry_at
job.status = "pending"  # still pending but not claimable until next_retry_at
job.claimed_by = None
job.claimed_at = None
job.started_at = None
```

Update claim filter to skip jobs not ready:

```python
# In claim():
from datetime import datetime
now = datetime.utcnow()

query = (
    select(Job)
    .where(Job.status == "pending")
    .where(Job.claimed_by.is_(None))
    .where(
        (Job.next_retry_at.is_(None)) | (Job.next_retry_at <= now)
    )
    .order_by(Job.priority.desc(), Job.created_at)
    .with_for_update(skip_locked=True)
    .limit(1)
```

## Task 3: Add checkpoint recording to JobQueue

**Files:** `src/worker/queue.py`

### Context

The `result_data` column already stores handler return value. We need to record **intermediate** checkpoints so a job can resume from a specific stage.

### Action

Add new method to `JobQueue`:

```python
async def record_checkpoint(
    self,
    db: AsyncSession,
    job: Job,
    checkpoint_id: str,
    checkpoint_data: dict[str, Any],
) -> None:
    """Record a named checkpoint for a running job.

    Stores intermediate progress so job can resume from this point
    if interrupted. Checkpoint data is merged into result_data
    under a 'checkpoints' key.

    Args:
        checkpoint_id: Unique identifier for this checkpoint (e.g., 'stage-3-crawl')
        checkpoint_data: Arbitrary dict with intermediate state
    """
    now = datetime.utcnow()

    # Initialize checkpoints structure
    if job.result_data is None:
        job.result_data = {}

    if "checkpoints" not in job.result_data:
        job.result_data["checkpoints"] = {}

    # Store checkpoint with timestamp
    job.result_data["checkpoints"][checkpoint_id] = {
        "data": checkpoint_data,
        "recorded_at": now.isoformat(),
        "attempt": job.attempt_count,
    }
    job.last_checkpoint = checkpoint_id

    event = JobEvent(
        id=uuid4(),
        job_id=job.id,
        event_type="checkpoint",
        message=f"Checkpoint recorded: {checkpoint_id}",
        extra={"checkpoint_id": checkpoint_id, "attempt": job.attempt_count},
        trace_id=job.trace_id,
    )
    db.add(event)
    await db.commit()

    broadcast_job_event({
        "event_type": "job_checkpoint",
        "job_id": str(job.id),
        "checkpoint_id": checkpoint_id,
    }, workspace_id=str(job.workspace_id) if job.workspace_id else None)

    logger.debug("checkpoint_recorded", job_id=str(job.id), checkpoint_id=checkpoint_id)
```

Also add `resume_from_checkpoint` method to extract checkpoint data:

```python
def get_checkpoint_data(job: Job, checkpoint_id: str) -> dict[str, Any] | None:
    """Extract checkpoint data from job result_data."""
    if job.result_data is None:
        return None
    checkpoints = job.result_data.get("checkpoints", {})
    checkpoint = checkpoints.get(checkpoint_id)
    if checkpoint:
        return checkpoint.get("data")
    return None
```

### Acceptance Criteria
- [ ] `record_checkpoint()` method exists and persists to DB
- [ ] Checkpoints stored in `result_data["checkpoints"]`
- [ ] `last_checkpoint` field updated on Job
- [ ] SSE event broadcast on checkpoint

### Context

When a job resumes from checkpoint, the handler needs to know WHERE to resume from. Currently handlers only receive `input_data`.

### Action

```python
context = {
    "job_id": str(job.id),
    "workspace_id": str(job.workspace_id),
    "trace_id": trace_id,
    "attempt": job.attempt_count,
}

# If resuming from checkpoint, add checkpoint data

    checkpoint_data = self.queue.get_checkpoint_data(job, job.last_checkpoint)
    context["resume_from_checkpoint"] = job.last_checkpoint
    context["checkpoint_data"] = checkpoint_data

# Call the handler

```
