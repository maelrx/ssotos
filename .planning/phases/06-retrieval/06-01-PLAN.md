## Task 3: Create Retrieval Pydantic Schemas

**Files created:** `src/schemas/retrieval.py` (new file)

### Context
- Phase 4 created `src/schemas/` but no `retrieval.py` exists yet
- Need schemas for search request/response and context pack per D-105

### Action
Create `src/schemas/retrieval.py` with these Pydantic v2 models:

```python
from pydantic import BaseModel, Field
from uuid import UUID
from datetime import datetime
from typing import Literal

# Search
class SearchRequest(BaseModel):
    q: str = Field(min_length=1)
    limit: int = Field(default=20, ge=1, le=100)
    mode: Literal["fts", "vector", "hybrid"] = "hybrid"

class SearchResult(BaseModel):
    chunk_id: UUID
    note_id: UUID
    note_path: str
    heading_path: str
    title: str
    tags: list[str]
    content_snippet: str = Field(description="Truncated to 500 chars")
    score: float
    score_breakdown: dict[str, float]
    fts_rank: int
    vector_rank: int
    why_matched: str

class SearchResponse(BaseModel):
    query: str
    mode: str
    results: list[SearchResult]
    total: int

# Context Pack
class ContextPackNeighbor(BaseModel):
    chunk_id: UUID
    heading_path: str
    content_snippet: str = Field(description="Truncated to 200 chars")
    chunk_index: int
    relevance_score: float

class ContextPackProvenance(BaseModel):
    note_path: str
    heading_path: str
    workspace_id: UUID
    indexed_at: datetime
    source_file: str

class ContextPackMetadata(BaseModel):
    kind: str
    tags: list[str]
    links: list[str]
    frontmatter: dict

class ContextPack(BaseModel):
    chunk_id: UUID
    note_reference: str = Field(description="Human-readable: note_path#heading")
    snippet: str = Field(description="The retrieved chunk content")
    heading_path: str
    score: float
    score_breakdown: dict[str, float]
    fts_rank: int
    vector_rank: int
    why_matched: str
    neighbors: list[ContextPackNeighbor]
    provenance: ContextPackProvenance
    metadata: ContextPackMetadata
```

### Acceptance Criteria
- [ ] `src/schemas/retrieval.py` created with all models above
- [ ] All models use Pydantic v2 (BaseModel, Field)
- [ ] SearchResult includes score breakdown (fts_component, vector_component, recency_boost if applicable)
- [ ] ContextPack includes all fields per D-105

## Task 4: Create RetrievalService

**Files created:** `src/services/retrieval_service.py`

### Context
- No `retrieval_service.py` exists yet
- Must implement FTS search, vector search, and RRF hybrid fusion per D-90, D-97

### Action
Create `src/services/retrieval_service.py` with:

1. **`EmbeddingsProvider` Protocol** — interface for embedding generation
2. **`OpenAIEmbeddingsProvider`** — `text-embedding-3-small` via httpx (per research §3.1)
3. **`RetrievalService`** class with methods:
   - `fts_search(workspace_id, query, limit)` → list of (chunk_id, fts_score, rank)
   - `vector_search(workspace_id, query_vector, limit)` → list of (chunk_id, cosine_sim, rank)
   - `hybrid_search(workspace_id, query, query_vector, limit, mode)` → list of RetrievalResult with RRF fusion
   - `build_context_pack(chunk_id, db)` → ContextPack with neighbors and provenance
4. **`reciprocal_rank_fusion(fts_results, vector_results, k=60)`** — per D-97

The service should use SQLAlchemy 2 async and the existing models (Chunk, Embedding, NoteProjection).

### Acceptance Criteria
