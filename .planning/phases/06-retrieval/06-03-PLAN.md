## Task 9: Job Handler — index_note

**Files modified:** `src/worker/handlers/index_note.py`

### Context
- Currently a placeholder that does nothing
- Must implement full chunking + FTS indexing + enqueue embeddings

### Action
Replace `handle_index_note` in `src/worker/handlers/index_note.py`:

```python
async def handle_index_note(input_data: dict) -> dict:
    """
    Index a single note: chunk it, store chunks with FTS vectors,
    enqueue embedding generation job.
    """
    note_id = input_data["note_id"]
    workspace_id = input_data["workspace_id"]
    operation = input_data.get("operation", "upsert")

    # 1. If delete: remove chunks and embeddings for this note
    if operation == "delete":
        # Delete embeddings for chunks of this note
        # Delete chunks
        # Update NoteProjection.indexed_at = None
        return {"note_id": str(note_id), "deleted": True}

    # 2. upsert: read note from filesystem
    note_proj = await db.get(NoteProjection, note_id)
    note_path = os.path.join(workspace.root_path, note_proj.note_path)
    markdown = Path(note_path).read_text(encoding="utf-8")

    # 3. Parse frontmatter (strip ---...---)
    frontmatter, body = parse_frontmatter(markdown)

    # 4. Chunk the content
    from src.services.chunking_service import markdown_to_chunks
    chunks = markdown_to_chunks(body, note_id)

    # 5. Delete old chunks + embeddings for this note
    # (delete embeddings first due to FK, then chunks)

    # 6. Insert new chunks with FTS tsvector
    for chunk in chunks:
        chunk.note_projection_id = note_id
        # Set content_tsv via raw SQL: to_tsvector('english', content)
        db.add(chunk)
    await db.flush()

    # 7. Update NoteProjection.indexed_at
    # 8. Enqueue generate_embeddings job for new chunk IDs
    await job_service.enqueue(
        "generate_embeddings",
        {"chunk_ids": [str(c.id) for c in chunks]},
        priority=5,
        workspace_id=workspace_id,
    )

    return {
        "note_id": str(note_id),
        "chunks_created": len(chunks),
        "enqueued_embeddings": len(chunks),
    }
```

### Acceptance Criteria
- [ ] `handle_index_note` handles `operation="delete"` to remove chunks and embeddings
- [ ] `operation="upsert"` reads markdown from filesystem using `note_path`
- [ ] Frontmatter is stripped before chunking
- [ ] Chunks are created with correct `heading_path`, `chunk_index`, `char_start`, `char_end`
- [ ] `content_tsv` tsvector is set via `to_tsvector('english', content)`
- [ ] `indexed_at` is updated on NoteProjection
- [ ] `generate_embeddings` job is enqueued with chunk IDs
- [ ] Old chunks/embeddings are deleted before re-inserting
