Create `src/core/policy/rules.py` with **PolicyRulesService** class for CRUD operations:

```python
class PolicyRulesService:
    def __init__(self, storage_path: Path | None = None)
    def list_rules(self) -> list[PolicyRule]
    def get_rule(self, rule_id: str) -> PolicyRule | None
    def create_rule(self, rule: PolicyRule) -> PolicyRule
    def update_rule(self, rule_id: str, updates: dict) -> PolicyRule | None
    def delete_rule(self, rule_id: str) -> bool
    def load_rules(self) -> list[PolicyRule]
    def save_rules(self, rules: list[PolicyRule]) -> None
    def reload_from_defaults(self) -> list[PolicyRule]
```

**Storage:**
- Store rules in `_system/policy/rules.yaml` (per Phase 1 _system/ folder structure)
- Load default rules if no rules file exists
- Each rule has a UUID-based id

**CRUD operations:**
- `list_rules()`: Returns all rules sorted by priority descending
- `get_rule(rule_id)`: Returns rule or None
- `create_rule(rule)`: Adds rule, saves, returns rule with id
- `update_rule(rule_id, updates)`: Updates rule fields, saves, returns updated rule
- `delete_rule(rule_id)`: Removes rule, saves, returns bool

**File operations:**
- `load_rules()`: Load from YAML, fallback to defaults
- `save_rules(rules)`: Write to YAML with proper serialization
- `reload_from_defaults()`: Replace all rules with defaults, save

Use yaml.safe_load/dump for serialization. Handle missing file gracefully.

Import Path from pathlib.
