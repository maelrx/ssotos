  <action>
Create `src/core/policy/service.py` with **PolicyService** class and **PolicyDeniedException**:

```python
class PolicyDeniedException(Exception):
    """Raised when a policy check denies an operation."""
    def __init__(self, result: PolicyResult):
        self.result = result
        super().__init__(f"Policy denied: {result.reason}")


class PolicyService:
    """Policy service wrapper that integrates with EventBus for audit logging.

    All mutations should go through check() before execution."""

    def __init__(self, rules_service: PolicyRulesService | None = None)
    def check(self, request: PolicyRequest) -> PolicyResult
    def check_or_raise(self, request: PolicyRequest) -> PolicyResult
```

**check_or_raise() logic:**
1. Call check() to evaluate policy
2. If outcome is DENY → raise PolicyDeniedException
3. If outcome is ALLOW_WITH_APPROVAL → raise PolicyDeniedException (approval not implemented in Phase 3)
4. Otherwise return result

**Audit logging (per D-46):**
- Import emit from core.events
- Import EventType from core.events
- Add new event types to EventType enum:
  - `POLICY_EVALUATED = "policy.evaluated"`
  - `POLICY_DENIED = "policy.denied"`
- On every check(), emit POLICY_EVALUATED with metadata: actor, capability_group, action, domain, path, outcome
- On DENY, emit POLICY_DENIED in addition

**Constructor:**
- If rules_service is None, create PolicyRulesService() with default storage
- Create PolicyEvaluator with loaded rules

This is the single integration point for all services.
  </action>
